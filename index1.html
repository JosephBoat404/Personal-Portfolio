Python Streamlit Project

This is a website that I built with Python and Streamlit. It allows you to calculate the value of Twitch's digital currency, Bits, in US dollars. You can enter the amount of Bits you have or want to buy, and the website will show you how much they are worth in dollars. This is useful for streamers and viewers who want to know how much they are spending or earning on Twitch.
